# simple-bass-boost
This is a simple one channel in-line bass boost circuit using the LM358 Op-amp. The circuit can be powered with around 5V. I made this circuit to enhance the sound of a cheap, no-name bluetooth speaker I had. 

I used this website as a reference for the design: https://electro-dan.co.uk/electronics/bassboost.aspx#single_supply_boost.
